# Node express server
This node application use :

- Atlas & Mongoose for MongoDb
- Express Js
- Jwt for token authentication
- Node

"dependencies": {

## How to use this project

Clone the repository

Go to the project folders

Execute commands:

```
npm i
npm start
```
